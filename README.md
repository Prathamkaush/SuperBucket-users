# Superbuket User Application

This folder is the standalone customer-facing Superbuket Expo application. It
contains only the user UI and does not depend on the renter or service-man app
folders.

## Run locally

```powershell
npm install
npm start
```

Use the Expo terminal options to open Android or iOS.

## Current UI

The demo includes splash, login, location selection, home, grocery, cart,
wallet, parcel delivery, print delivery, PennyWorks, rentals, marketplace,
order tracking, notifications, and profile screens.

The screens currently use local UI data and placeholders. Backend integration,
authentication, payments, uploads, and live order tracking are not included in
this template yet.
